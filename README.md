# Online-Shopping-System
## Overview
This Java Swing application simulates an online shopping platform.  
Users can browse products, add them to the cart, and place orders.

## Features
- Browse product catalog  
- Add products to cart  
- Place orders  
- View order history  

## Technologies
- Java  
- Swing (GUI)
